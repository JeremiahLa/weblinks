# Jeremiah Lalisan Link Hub

This project is a simple personal landing page that showcases a profile image, a short introduction, and links to several online profiles such as Roblox, Steam, GitHub, and Spotify.

## What it does

- Displays a clean single-page layout for a personal profile
- Includes clickable buttons for social and gaming links
- Supports a dark/light theme toggle for the page
- Uses HTML and CSS with a small bit of JavaScript for interactivity

## Files

- index.html: The main page content and structure
- styles.css: Styling for the page layout and theme

## How to view it

1. Open the project folder in a browser.
2. Double-click index.html, or open it from your browser's File menu.

## Notes

This is a lightweight static website, so no installation or build step is required.
